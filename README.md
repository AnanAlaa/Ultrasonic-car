# Ultrasonic Car Project

This project is an Arduino-based car that uses an Ultrasonic sensor to detect and avoid obstacles.

## Features:
- Obstacle detection
- Automatic redirection
- Basic embedded system application

## Tools Used:
- Arduino Uno
- Ultrasonic Sensor
- Motor Driver (L298N)
- Batteries
- Wires and chassis
